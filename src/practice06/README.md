# Practice 06 — 출고 우선순위 정하기

## 상황
물류센터장이 말합니다.

> "주문이 밀리고 있어. 주문일이 오래된 것부터 먼저 출고해.
> 같은 날이면 금액 높은 것 우선이야. 정렬된 목록 뽑아줘."

## 목표
- `Comparable` 인터페이스를 구현하여 객체를 정렬할 수 있다
- `Comparator`를 사용하여 다양한 기준으로 정렬할 수 있다
- `Collections.sort()`를 사용할 수 있다

## 요구사항
1. 주문 리스트를 **주문일 오름차순**(오래된 것 먼저)으로 정렬하여 출력하시오
2. 주문 리스트를 **상품금액 내림차순**(비싼 것 먼저)으로 정렬하여 출력하시오
3. 주문 리스트를 **브랜드별 오름차순 → 같은 브랜드 내에서 금액 내림차순**으로 정렬하시오
4. (추가) 상위 5건만 출력하시오

## 힌트
- 문자열 비교: `order1.getOrderDate().compareTo(order2.getOrderDate())`
- 숫자 비교(내림차순): `order2.getPrice() - order1.getPrice()`
- 정렬: `Collections.sort(orders, comparator)`
- Comparator 람다: `(o1, o2) -> o1.getOrderDate().compareTo(o2.getOrderDate())`
- 복합 정렬:
  ```java
  Comparator.comparing(Order::getBrand)
            .thenComparing(Comparator.comparingInt(Order::getPrice).reversed())
  ```
- `import java.util.Collections;`와 `import java.util.Comparator;`를 잊지 마세요
