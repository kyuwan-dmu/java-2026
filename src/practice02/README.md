# Practice 02 — 전체 주문 한눈에 보기

## 상황
팀장님이 말합니다.

> "주문 하나만 보면 뭐해, 오늘 들어온 거 전부 목록으로 뽑아봐.
> 총 몇 건인지도 알려줘."

주문이 74건입니다. 하나씩 변수로 만들 수는 없으니 배열과 반복문을 사용합니다.

## 목표
- 배열에 담긴 여러 객체를 `for`문으로 순회할 수 있다
- 향상된 for문(for-each)을 사용할 수 있다
- 반복문 안에서 객체의 메서드를 호출하여 원하는 형식으로 출력할 수 있다

## 요구사항
1. `OrderDataProvider.getOrdersAsArray()`로 전체 주문을 받아오시오
2. 기본 for문(`for(int i=0; ...)`)으로 전체 주문을 아래 형식으로 출력하시오
3. 향상된 for문(`for(Order order : orders)`)으로 동일하게 출력하시오
4. 전체 주문 건수를 출력하시오

## 출력 예시
```
===== 전체 주문 목록 =====
 1. [바디홀릭] 스테이누디 디스커버리 세트 (20ml*6종) - 20,000원
 2. [바디홀릭] 스테이누디 디스커버리 세트 (20ml*6종) - 20,000원
 3. [바디홀릭] 스테이누디 디스커버리 세트 (20ml*6종) - 20,000원
 4. [바디홀릭] 스테이누디 라인 바디미스트 6종 - 12,900원
 ...
========================
총 74건
```

## 힌트
- 배열의 길이는 `orders.length`로 알 수 있습니다
- 번호를 붙이려면 기본 for문에서 `i + 1`을 출력합니다
- `order.toString()`은 `System.out.println(order)` 시 자동 호출됩니다
