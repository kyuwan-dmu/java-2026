# Practice 03 — 조건에 맞는 주문 찾기

## 상황
사수가 말합니다.

> "바디홀릭 브랜드 주문만 먼저 포장 준비해. 몇 건이고 금액이 얼마인지도 정리해줘."

74건 전부를 포장할 수는 없으니, 조건에 맞는 것만 골라내야 합니다.

## 목표
- 반복문 안에서 `if`문으로 조건을 걸어 원하는 데이터만 필터링할 수 있다
- 조건에 맞는 항목의 개수를 셀 수 있다
- 조건에 맞는 항목의 합계를 구할 수 있다

## 요구사항
1. 브랜드가 **"바디홀릭"**인 주문만 출력하시오
2. 해당 주문의 총 건수를 출력하시오
3. 해당 주문의 상품금액 합계를 출력하시오
4. (추가) 상품금액이 **15,000원 이상**인 주문만 출력하시오

## 출력 예시
```
===== 바디홀릭 주문 목록 =====
[바디홀릭] 스테이누디 디스커버리 세트 (20ml*6종) - 20,000원 (경기 안산시)
[바디홀릭] 스테이누디 디스커버리 세트 (20ml*6종) - 20,000원 (울산 남구)
[바디홀릭] 스테이누디 디스커버리 세트 (20ml*6종) - 20,000원 (경북 안동시)
...
============================
바디홀릭 주문: 65건
합계: 1,064,000원

===== 15,000원 이상 주문 =====
...
```

## 힌트
- 문자열 비교는 `order.getBrand().equals("바디홀릭")`을 사용합니다
- 합계 변수는 for문 **바깥**에 선언합니다: `int totalPrice = 0;`
- 조건을 여러 개 조합하려면 `&&` 연산자를 사용합니다
