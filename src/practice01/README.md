# Practice 01 — 주문 데이터 확인하기

## 상황
당신은 오늘부터 쇼핑몰 물류팀 인턴으로 출근했습니다.
팀장님이 말합니다.

> "이게 오늘 처리할 주문 데이터야. 일단 하나 꺼내서 내용 확인해봐."

## 목표
- `Order` 클래스의 구조를 이해한다
- 객체를 생성하고 getter를 사용하여 데이터를 출력할 수 있다
- `System.out.println`으로 원하는 형식의 출력을 만들 수 있다

## 요구사항
1. `OrderDataProvider.getOrdersAsArray()`로 주문 데이터를 받아오시오
2. 배열의 **첫 번째** 주문을 변수에 담으시오
3. 해당 주문의 모든 정보를 아래 형식으로 출력하시오

## 출력 예시
```
===== 주문 상세 =====
주문번호: 2596171645
상품명: [단독]"니치 향" 바디홀릭 스테이누디 디스커버리 세트 (20ml*6종)
브랜드: 바디홀릭
옵션: (없음)
수량: 1
금액: 20,000원
주문일: 2024-08-27 16:53:41
상태: 301 배송 준비 중
지역: 경기 안산시
====================
```

## 힌트
- `Order[] orders = OrderDataProvider.getOrdersAsArray();`로 데이터를 받아옵니다
- `orders[0]`으로 첫 번째 주문에 접근합니다
- `order.getProductName()` 처럼 getter를 사용합니다
- 금액에 콤마를 넣으려면 `String.format("%,d", price)`를 사용합니다
- 옵션이 빈 문자열이면 "(없음)"으로 출력해보세요
