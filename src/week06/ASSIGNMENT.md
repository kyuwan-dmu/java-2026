# 6주차 과제: 학생 성적 관리 시스템

## 개요

우리 학교 성적 관리 시스템을 만들려고 합니다.
기존에 사용하던 `Student` 클래스를 확장하여, 성적 관리와 리포트 기능을 갖춘 콘솔 프로그램을 구현하세요.

## 제출 방법

1. `main` 브랜치에서 자신의 브랜치를 생성한다
    - 브랜치 이름: `feat/학번` (예: `feat/20260001`)
2. 아래 요구사항을 구현한다
3. 작업이 완료되면 `main` 브랜치로 **Pull Request**를 올린다
    - PR 제목: `학번_이름`
    - PR 본문: 어떻게 구현하였는지 자유롭게 작성

---

## 요구사항

### 학생

- 학생은 이름과 점수를 가집니다.
- 점수는 0~100 사이만 유효합니다. 범위를 벗어난 점수를 세팅하려고 하면 거부되어야 합니다.
- 점수에 따라 등급이 결정됩니다: 90 이상 `A`, 80 이상 `B`, 70 이상 `C`, 나머지 `F`
- 학생 객체를 만드는 방법은 여러 가지가 있어야 합니다:
    - 이름과 점수를 모두 지정
    - 이름만 지정 (점수는 0)
    - 아무것도 지정하지 않음 (이름은 "이름없음", 점수는 0)
- 지금까지 총 몇 명의 학생이 생성되었는지 추적할 수 있어야 합니다.
- 외부에서 학생의 필드에 직접 접근할 수 없어야 합니다.

### 점수 검증

- 점수가 합격(60점 이상)인지 판별하는 기능이 필요합니다.
- 이 기능은 학생 클래스와 같은 패키지 내에서만 사용 가능하고, 외부 패키지에서는 접근할 수 없어야 합니다.

### 성적 리포트

- 리포트 기능은 학생과 **다른 패키지**에 위치해야 합니다.
- 객체를 생성하지 않고도 리포트 기능을 사용할 수 있어야 합니다.
- 다음 기능을 제공합니다:
    - 학생 2명의 평균 점수 계산
    - 학생 배열의 평균 점수 계산 (위와 같은 이름의 메서드로)
    - 학생 배열 전체에 보너스 점수 일괄 부여
    - 학생 배열 전체 정보 출력

## 프로그램 동작

프로그램을 실행하면 아래와 같은 메뉴가 표시되고, 번호를 입력하여 기능을 실행합니다.
`0`을 입력하면 프로그램이 종료됩니다.

```
===== 성적 관리 시스템 =====
1. 학생 등록
2. 전체 학생 목록 조회
3. 학생 점수 수정
4. 보너스 점수 일괄 부여
5. 평균 점수 조회
6. 총 학생 수 조회
0. 종료
========================
선택> 
```

### 1. 학생 등록

학생을 등록합니다. 이름과 점수를 입력받습니다.

```
선택> 1
이름 입력 (없으면 Enter): 김철수
점수 입력 (없으면 Enter): 85
등록 완료: 김철수 (85점)
```

```
선택> 1
이름 입력 (없으면 Enter): 이영희
점수 입력 (없으면 Enter): 
등록 완료: 이영희 (0점)
```

```
선택> 1
이름 입력 (없으면 Enter): 
점수 입력 (없으면 Enter): 
등록 완료: 이름없음 (0점)
```

### 2. 전체 학생 목록 조회

등록된 모든 학생의 이름, 점수, 등급을 출력합니다.

```
선택> 2
이름: 김철수, 점수: 85, 등급: B
이름: 이영희, 점수: 0, 등급: F
이름: 이름없음, 점수: 0, 등급: F
```

### 3. 학생 점수 수정

학생 이름을 입력받아 해당 학생의 점수를 수정합니다.
유효하지 않은 점수(0~100 범위 밖)를 입력하면 거부됩니다.

```
선택> 3
이름 입력: 이영희
새 점수 입력: 150
유효하지 않은 점수입니다

선택> 3
이름 입력: 이영희
새 점수 입력: 92
점수 수정 완료: 이영희 (92점)
```

### 4. 보너스 점수 일괄 부여

모든 학생에게 동일한 보너스 점수를 부여합니다.

```
선택> 4
보너스 점수 입력: 5
보너스 부여 완료!

선택> 2
이름: 김철수, 점수: 90, 등급: A
이름: 이영희, 점수: 97, 등급: A
이름: 이름없음, 점수: 5, 등급: F
```

### 5. 평균 점수 조회

등록된 전체 학생의 평균 점수를 출력합니다.

```
선택> 5
전체 평균: 64.0
```

### 6. 총 학생 수 조회

```
선택> 6
총 학생 수: 3
```

### 0. 종료

```
선택> 0
프로그램을 종료합니다.
```
