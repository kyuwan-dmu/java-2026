# Practice 05 — 매출 보고서 만들기

## 상황
팀장님이 퇴근 전에 말합니다.

> "오늘 오전 주문 데이터로 간단한 보고서 만들어줘.
> 브랜드별 주문 건수랑 매출, 그리고 전체 평균 주문 금액 알려줘."

## 목표
- 리스트를 순회하며 조건별로 데이터를 그룹핑할 수 있다
- 합계, 평균 등 집계 로직을 구현할 수 있다
- 결과를 보기 좋은 형식으로 출력할 수 있다

## 요구사항
1. **전체 매출** (상품금액 합계)을 출력하시오
2. **전체 평균 주문 금액**을 출력하시오
3. **브랜드별** 주문 건수와 매출 합계를 출력하시오
4. **지역별** 주문 건수를 출력하시오 (시/도 단위, 예: "경기", "서울", "부산" ...)
5. (추가) 가장 매출이 높은 브랜드를 찾아 출력하시오

## 출력 예시
```
===== 매출 보고서 =====
전체 주문: 74건
전체 매출: 1,264,000원
평균 주문 금액: 17,081원

--- 브랜드별 ---
바디홀릭: 65건 / 1,106,400원
머지: 9건 / 157,600원

--- 지역별 주문 건수 ---
경기: 12건
서울: 8건
부산: 7건
...
========================
```

## 힌트
- 브랜드가 2종류뿐이므로 각각 변수를 만들어도 됩니다
  - `int bodyHolicCount = 0; int bodyHolicTotal = 0;`
- 지역별 집계는 도전적입니다. 지역명의 첫 단어만 비교해보세요
  - `order.getRegion().startsWith("경기")`
- 평균은 `총합 / 건수`입니다. 정수끼리 나누면 소수점이 사라지니 주의!
  - `(double) totalPrice / count`
